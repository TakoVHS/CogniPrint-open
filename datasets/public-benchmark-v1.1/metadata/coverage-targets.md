# Public Benchmark v1.1 Coverage Targets

## Objective

Use the next benchmark increment to reduce the current concentration around a small number of literary baselines and a narrow language spread.

## Coverage targets

- `20` to `50` public baselines
- `100` to `300` controlled variants
- at least `4` languages in the next increment if provenance remains clean
- at least `3` source domains in the next increment

## Priority mix

Priority target order:

1. more English non-literary baselines
2. more French and Spanish baselines
3. first German baseline
4. first Russian baseline
5. optional manually authored Vietnamese sample only with explicit release intent

## Release balance checks

Before release, verify:

- no single language dominates the new increment disproportionately;
- no single source domain dominates the new increment disproportionately;
- every released baseline has a stable provenance record;
- every released variant maps to a released baseline cleanly.
