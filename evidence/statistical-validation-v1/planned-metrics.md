# Statistical Validation v1 Planned Metrics

Historical note: this file is superseded as a current-status source by
`validation/wave005-descriptive-validation/`,
`validation/statistical-validation-v1.2/`, `validation/inferential-v1/`, and
`docs/current-scientific-status-2026-05-22.md`. It remains in the repository as
a planning note for the earlier `statistical-validation-v1` layer.

This note operationalizes the current statistical validation plan into concrete output classes.

## Planned metrics

- effect-size summaries across perturbation types
- bootstrap confidence intervals for selected similarity and profile-shift measures
- within-campaign variance summaries
- between-campaign variance summaries
- random perturbation baseline comparison
- sensitivity to stability thresholds

## Planned input layers

- campaign-level empirical evidence under `evidence/empirical-v1/`
- released public benchmark subset under `evidence/public-benchmark-v1/`

The original implementation pass described by this file has been followed by
later descriptive and pre-review inferential candidate artifacts. Do not use
this note as the current repository status for statistical evidence or
