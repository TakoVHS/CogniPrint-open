# Methods Summary

This file summarizes the public methods layer for the CogniPrint empirical evidence snapshot v1.

## Campaign design

The current workflow uses controlled perturbation campaigns. Each campaign compares a baseline text against a light edit and one or more controlled variants. The purpose is to observe how compact statistical profiles respond to bounded changes in wording, syntax, style, segmentation, or rendering.

## Current empirical base

- 5 controlled perturbation campaigns
- 41 comparison rows
- campaign-004 contribution: 1 controlled series / 11 comparison rows

## Outputs recorded by the workstation

The local workstation records:

- campaign manifests;
- campaign result JSON files;
- campaign result CSV files;
- campaign summaries;
- manuscript appendix drafts;
- LaTeX-oriented tables;
- clean Methods, Results, and Limitations drafting files.

## Provenance handling

Input source records are maintained separately from generated summaries. Raw local texts are not included in this public snapshot. This snapshot contains public summaries, counts, and interpretation limits only.

## Interpretation

The method supports review of observed profile shifts, measured metric deltas, perturbation effects, and stability tendencies inside the current controlled corpus. It does not establish broad generalization beyond that corpus.
