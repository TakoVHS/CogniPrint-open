# CogniPrint Empirical Evidence Snapshot v1

This directory is the public, stable summary snapshot for the current CogniPrint empirical evidence package.

## Status

CogniPrint currently has a working empirical evidence package supporting a follow-up manuscript. This snapshot is intended for colleague review, due diligence, and manuscript consolidation. It should not be presented as a publication-ready empirical paper.

## Current empirical base

- 5 controlled perturbation campaigns
- 41 comparison rows
- campaign-004 contribution: 1 controlled series / 11 comparison rows
- provenance-linked input records
- dataset scaffolds
- multi-campaign summaries
- clean Methods, Results, and Limitations drafting materials

## What is included here

This directory contains public summaries and metadata only:

- `manifest.json` — machine-readable snapshot manifest
- `counts.json` — machine-readable empirical counts
- `methods-summary.md` — public methods summary
- `results-summary.md` — public results summary
- `limitations-summary.md` — public limitations summary
- `evidence-table.md` — compact evidence table
- `provenance-summary.md` — public provenance summary

## What is not included here

Raw local input texts are not included in this snapshot. The public snapshot summarizes campaign structure, counts, methods, outputs, and interpretation limits without exposing local-only raw material.
