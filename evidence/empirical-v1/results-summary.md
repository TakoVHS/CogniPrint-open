# Results Summary

This file summarizes the public results layer for the CogniPrint empirical evidence snapshot v1.

## Current evidence package

The current evidence package contains:

- 5 controlled perturbation campaigns;
- 41 comparison rows;
- campaign-004 contribution of 1 controlled series and 11 comparison rows;
- provenance-linked input records;
- dataset scaffolds;
- campaign-level summaries;
- clean Methods, Results, and Limitations drafting materials.

## What the results show

The current outputs document observed profile behavior under controlled text perturbations. They support discussion of:

- observed patterns across campaign outputs;
- measured shifts between baseline and variant profiles;
- profile differences under bounded edits;
- perturbation effects inside the current corpus;
- stability tendencies that can be reviewed by colleagues.

## What the results do not show

The current results do not establish broad generalization across heterogeneous public corpora. They also do not provide a source decision, legal conclusion, or general-purpose classification result.

## Review interpretation

The results should be read as a working empirical evidence package supporting a follow-up manuscript. The correct next step is manuscript consolidation and external review, not stronger claims from the current corpus alone.
