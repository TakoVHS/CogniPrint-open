# Limitations Summary

This file summarizes the public limitations layer for the CogniPrint empirical evidence snapshot v1.

## Scope limits

The current evidence package is intentionally bounded. It records behavior inside the present controlled perturbation campaigns and should not be interpreted as broad corpus-level validation.

## Current limitations

- The current empirical base is still small.
- The current evidence package summarizes 5 controlled perturbation campaigns and 41 comparison rows.
- Campaign-004 contributes 1 controlled series and 11 comparison rows.
- The current texts are primarily local research material rather than a broad public benchmark corpus.
- The current results document profile behavior only.
- The current package supports manuscript development and colleague review, not publication-ready conclusions.

## Required future work

Stronger empirical maturity requires:

- larger corpora;
- more heterogeneous text domains;
- public mini-benchmark sets;
- additional perturbation families;
- statistical validation layers such as effect-size summaries or confidence intervals;
- external review and issue-based feedback.

## Interpretation guardrail

The correct description is: working empirical evidence package supporting a follow-up manuscript.
