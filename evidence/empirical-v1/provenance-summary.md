# Provenance Summary

This file summarizes public provenance handling for the CogniPrint empirical evidence snapshot v1.

## Principle

The public evidence snapshot separates public summaries from local raw input material. It records what was summarized without publishing private or local-only raw texts.

## Current state

The current empirical package includes:

- provenance-linked input records;
- dataset scaffolds;
- copied source metadata where appropriate;
- public summaries and counts;
- no raw local input texts in this public snapshot.

## Public snapshot contents

The public snapshot includes:

- machine-readable counts;
- public methods summary;
- public results summary;
- public limitations summary;
- public evidence table;
- public provenance summary.

## Excluded materials

The following are intentionally excluded from this public snapshot unless separately reviewed for release:

- raw local input texts;
- private experimental notes;
- temporary run outputs;
- local-only workspace materials;
- ambiguous or sensitive source material.

## Review use

This provenance summary is intended to help colleagues understand the boundary between public evidence artifacts and local execution materials.
